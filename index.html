Experiência

Desenvolvedor Mobile - Maui.Net

NET MAUI (Multi-platform App UI)

Multi-platform é uma abordagem moderna e eficiente para criar aplicativos multiplataforma, utilizando a linguagem de programação C# e a plataforma .NET. Essa tecnologia é uma evolução do Xamarin.Forms, projetada para simplificar ainda mais o processo de desenvolvimento e fornecer uma experiência de usuário consistente em diferentes dispositivos e sistemas operacionais, como Android, iOS, Windows e macOS.

Desenvolvedor Web

PHP e ASP.Net

PHP (Hypertext Preprocessor) é uma linguagem de script de servidor de código aberto amplamente utilizada para o desenvolvimento web. Criada originalmente em 1994 por Rasmus Lerdorf, PHP é especialmente adequada para o desenvolvimento de aplicações web dinâmicas e interativas. Como linguagem de script do lado do servidor, o PHP é executado no servidor web antes de enviar o HTML resultante para o navegador do usuário.

ASP.NET é uma estrutura de desenvolvimento web desenvolvida pela Microsoft, projetada para criar aplicativos web dinâmicos e robustos. Lançado inicialmente em 2002, ASP.NET é uma das principais escolhas para desenvolvimento web na plataforma Windows.

Cloud Computing

Azure

Azure Cloud Computing é uma plataforma de computação em nuvem desenvolvida pela Microsoft, oferecendo uma ampla gama de serviços para hospedar, gerenciar e desenvolver aplicativos e serviços de maneira eficiente e escalável. Lançado em 2010, o Azure se tornou uma das principais opções para empresas que buscam aproveitar os benefícios da computação em nuvem.

DevOps

Azure DevOps

Azure DevOps é uma plataforma de serviços da Microsoft que oferece um conjunto abrangente de ferramentas para ajudar as equipes de desenvolvimento de software a colaborar de forma eficaz, automatizar processos e entregar software de alta qualidade de maneira mais rápida e eficiente. Lançado inicialmente como Visual Studio Team Services (VSTS) em 2015, foi posteriormente renomeado para Azure DevOps em 2018, refletindo sua integração com o ecossistema do Microsoft Azure.
